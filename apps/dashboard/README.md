# N0Tune Dashboard

The dashboard shows service status and lets you manage memories, style profile, documents, context preview, semantic cache, and security notes.

Run locally:

```powershell
npm install
npm run dev --workspace apps/dashboard
```

Open:

```text
http://localhost:3000
```

The dashboard expects the API at `NEXT_PUBLIC_N0TUNE_API_BASE_URL`, defaulting to `http://localhost:8000`.
