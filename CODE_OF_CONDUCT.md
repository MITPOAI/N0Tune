# Code of Conduct

N0Tune follows a simple contributor standard: be respectful, be precise, and keep discussions focused on improving the project.

## Expected behavior

- Use welcoming and inclusive language.
- Respect different experience levels and viewpoints.
- Give actionable technical feedback.
- Accept maintainers' decisions when they are made for project scope, safety, or quality.
- Disclose conflicts of interest where relevant.

## Unacceptable behavior

- Harassment, threats, or personal attacks.
- Publishing private information without permission.
- Sexualized language or imagery.
- Repeated disruptive behavior after maintainers ask for it to stop.
- Security testing that accesses, modifies, deletes, or exposes data that does not belong to you.

## Enforcement

Maintainers may remove comments, close issues, block accounts, or revoke contribution access when behavior harms the project or community.

Report conduct issues privately to the repository owner. For security issues, follow [SECURITY.md](SECURITY.md).
